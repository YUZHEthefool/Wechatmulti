# 微信多开助手

一个用于解除微信多开限制的 Windows 工具。

## 原理

微信通过创建命名互斥锁（Mutex）来防止多开。本工具通过枚举系统句柄找到该 Mutex 并关闭它，从而允许启动多个微信实例。

## 使用方法

### 方法一：手动执行（推荐）

1. 启动第一个微信
2. **以管理员身份**运行 `wechatmult.exe`
3. 选择 `1. 单次执行`
4. 现在可以启动更多微信实例了

### 方法二：配合 VBS 脚本

1. 编辑 `启动微信.vbs`，修改 `wechatPath` 为你的微信安装路径
2. 双击 VBS 脚本启动微信
3. 脚本会自动调用工具解除限制

## 命令行参数

| 参数 | 说明 |
|------|------|
| （无参数） | 显示交互式菜单 |
| `--kill-mutex-all` | 静默关闭所有微信的互斥锁后退出 |
| `--kill-mutex <pid>` | 关闭指定 PID 进程的互斥锁 |

## 编译

需要 Rust 工具链和 MSVC 构建工具。

```bash
cargo build --release
```

编译产物位于 `target/release/wechatmult.exe`

## 注意事项

- 必须以**管理员权限**运行
- 仅支持 Windows 系统
- 需要先启动微信，再运行本工具
